
###Project includes:
* Require.js

* Underscore
* KendoUI
* Almond - optimizer
* Moment - calender and time organizer

* uglify.js

* typedArray.js - polyfills for backward browser compatibility
* Blob.js - providing support for IE < 9
* jquery-table2excel.js - Excel xls export
